# SQL-аналитика: бразильский e-commerce Olist

Учебный проект по аналитическому SQL на реальном датасете публичных заказов
бразильского маркетплейса Olist (~100 тыс. заказов за 2016–2018 гг.).

## Стек и техники

- **СУБД:** PostgreSQL
- **Техники SQL:** JOIN, оконные функции (LAG, RANK, SUM OVER),
  агрегации (SUM/AVG/COUNT, HAVING), CTE, условная логика (CASE).

## Датасет

Данные не включены в репозиторий из-за размера. Для воспроизведения результатов, скачайте пожалуйста датасет с Kaggle:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Датасет состоит из 9 связанных таблиц: заказы, клиенты, товары, продавцы,
позиции заказов, платежи, отзывы, геолокация и справочник перевода категорий.

## Как воспроизвести

1. Создайте базу данных в PostgreSQL.
2. Выполните `downloading_data/01_create_tables.sql` — создание таблиц.
3. В `downloading_data/02_load_data.sql` замените путь на папку с вашими CSV и выполните — загрузка данных.
4. Запускайте запросы из `queries/` — каждый файл решает один бизнес-вопрос.

## Ключевые выводы

**Сезонность и рост выручки (q01).** Ярко выражен пик в ноябре 2017 -
месячная выручка выросла на ~340 тыс. (эффект Чёрной пятницы). Сразу за пиком
последовал резкий спад в декабре (~−262 тыс.), что говорит о выраженной
сезонности спроса.

**Средний чек стабилен (q03).** AOV держится в районе 140 на протяжении всего
периода. Значит рост общей выручки обеспечивается увеличением числа заказов,
а не размера чека(растёт охват, а не средний доход с клиента).

**Категории-лидеры (q04, q10).** Наибольшую выручку приносят health_beauty,
watches_gifts и bed_bath_table. ABC-анализ подтверждает: небольшая группа
категорий (группа A) формирует основную часть выручки.

**Способы оплаты (q06).** Кредитная карта абсолютно доминирует - 78% выручки,
причём в среднем 3,5 платежа, то есть покупатели активно пользуются рассрочкой.
Boleto (местный платёжный метод) - второй по популярности с ~18%.

**Удержание клиентов - слабое место (q09).** Лишь 3,1% клиентов совершают
повторные покупки; 96,9% покупают однократно. Это ключевая точка роста.

**Качество доставки (q05).** Средний срок доставки - ~12,6 дня, при этом
доля опозданий относительно обещанной даты низкая (~8%). Логистика в целом
надёжна: маркетплейс закладывает реалистичные сроки с запасом.

**Оценки покупателей (q07).** Самые высокие средние оценки - у категорий книг
(books_general_interest, books_technical) и товаров для дома/еды. Это категории
с предсказуемым качеством и низким уровнем разочарований.

## Итог

Бизнес растёт за счёт привлечения новых клиентов и сезонных всплесков, но почти
не удерживает покупателей. Основные рычаги роста: программы повторных покупок
(лояльность) и фокус на высокомаржинальных категориях группы A.
